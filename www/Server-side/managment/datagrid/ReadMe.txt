It's very simple to get started!!!


1. Copy code_template.php file to needed directory on your server.

2. Copy folder named "datagrid" (with all includes files) in the same 
   directory on your server.

3. Create database on your database server.

4. Change the code in code_template.php file according to your needs: 
   4.1 Uncomment all needed lines commented by '//'. 
   4.2 Change all needed variable values on yours.

5. Rename code_template.php file according to your needs.

6. Enjoy!
